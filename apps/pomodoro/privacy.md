# Privacy Policy

**Pomodoro — Focus Timer**
*Last updated: April 4, 2026*

## Data Collection

This application does not collect, store, or transmit any personal data.

## What We Do NOT Collect

- No personal information
- No usage analytics
- No crash reports
- No location data
- No device identifiers
- No advertising data

## Network Access

This application makes zero network requests. It works entirely offline.

## Permissions

The app requests only the following permissions:

- **VIBRATE** — Used for haptic feedback when pressing the timer button
- **POST_NOTIFICATIONS** — Used to notify you when the timer completes

No other permissions are requested or required.

## Data Storage

Timer state is held in memory only and is not persisted between sessions. No data is written to disk, cloud storage, or any external service.

## Third-Party Services

This application does not integrate with any third-party services, SDKs, analytics platforms, or advertising networks.

## Children's Privacy

This application does not knowingly collect information from children under the age of 13. The app contains no user-generated content, social features, or in-app purchases.

## Changes to This Policy

If we update this privacy policy, the changes will be posted on this page with an updated revision date.
